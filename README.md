# Hotel Management System

This project is designed to automate and enhance hotel management processes. The main goal is to simplify tasks such as user registration, room management, client tracking, and booking, reducing the time staff spend on manual data handling.

---

## 🚀 Features

1. **Staff Registration and Login**: Secure authentication for staff members.  
2. **Admin Panel**: A centralized panel for administrators to manage the system.  
3. **Room Management**: Easily manage rooms, including their status, price, and availability.  
4. **Booking Management**: Staff can quickly book rooms for clients and track reservations.  

---

## 🛠 Problem Solved

The application addresses common inefficiencies in hotel operations, such as managing processes on paper or spreadsheets. With this software, hotels can:

1. **Enhance Data Security**: Administrators can control user access and permissions.  
2. **Improve Booking Efficiency**: Staff can track room availability, prices, and statuses in real time, streamlining the booking process.  
3. **Monitor Financial Performance**: Generate reports to track daily and overall profits.  

---

## 📋 Requirements

- **.NET Framework**: Version 8.0 or higher  
- **Database**: Microsoft SQL Server

---

1. Clone the repository:  
   ```bash
   git clone https://github.com/tosentai/CSharp_Final_1.5year.git
   ```
